## О проекте
------
Одностранничный сайт для [Яндекс.Практикум](https://practicum.yandex.ru/) о разных методах обучения.
В работе применяются:
- Флексбокс-вёрстка
- Позиционирование блоков и элементов
- Анимации
- API Youtube
и другие методы.
